常见抗生素使用指南

一、抗生素分类与作用机制

1. β-内酰胺类
包括青霉素类和头孢菌素类，通过抑制细菌细胞壁合成发挥杀菌作用。

青霉素类：
- 青霉素G：主要用于链球菌、肺炎球菌感染
- 阿莫西林：广谱青霉素，用于上呼吸道感染、尿路感染、幽门螺杆菌感染
- 阿莫西林/克拉维酸：增强对产β-内酰胺酶菌株的抗菌活性

头孢菌素类（按代际分）：
- 第一代（头孢氨苄、头孢唑林）：主要针对革兰阳性菌
- 第二代（头孢呋辛、头孢克洛）：对革兰阴性菌活性增强
- 第三代（头孢曲松、头孢他啶）：对革兰阴性菌活性强，可穿透血脑屏障
- 第四代（头孢吡肟）：广谱，对ESBL产酶菌部分有效

2. 大环内酯类
通过抑制核糖体50S亚基阻断蛋白质合成。
- 阿奇霉素：社区获得性肺炎、支原体感染、衣原体感染
- 克拉霉素：幽门螺杆菌根除方案组成药物
- 红霉素：青霉素过敏患者的替代选择

3. 喹诺酮类
通过抑制DNA回旋酶和拓扑异构酶IV发挥杀菌作用。
- 左氧氟沙星：呼吸道感染、泌尿系统感染
- 莫西沙星：社区获得性肺炎、急性细菌性鼻窦炎
- 注意：有肌腱断裂风险，不推荐18岁以下使用

4. 氨基糖苷类
通过不可逆结合30S核糖体亚基抑制蛋白合成。
- 庆大霉素、阿米卡星
- 主要用于革兰阴性菌严重感染
- 需监测肾功能和听力（耳肾毒性）

5. 碳青霉烯类
超广谱β-内酰胺类，用于多重耐药菌感染。
- 美罗培南、亚胺培南/西司他丁
- 作为"最后防线"抗生素，应严格控制使用

二、抗生素使用基本原则

1. 明确感染诊断：尽量在使用前采集标本进行病原学检查。
2. 经验性用药：在病原学结果回报前，根据感染部位、社区或院内感染、当地耐药性数据选择。
3. 目标性治疗：根据药敏结果调整用药，选择窄谱、有效的抗生素。
4. 疗程规范：避免疗程过短（治疗失败/耐药）或过长（不良反应/耐药）。
5. 联合用药指征：严重感染、混合感染、需要协同杀菌效应时。

三、常见感染经验性用药方案

上呼吸道感染：
- 急性咽炎/扁桃体炎（A群链球菌）：阿莫西林 或 青霉素V
- 急性中耳炎：阿莫西林（首选），失败时用阿莫西林/克拉维酸

社区获得性肺炎（CAP）：
- 轻症门诊患者：阿莫西林 或 阿奇霉素/多西环素
- 中重症住院患者：β-内酰胺类 + 大环内酯类，或呼吸喹诺酮类单药
- 重症ICU患者：β-内酰胺类 + 大环内酯类 或 喹诺酮类

泌尿系统感染：
- 单纯性膀胱炎：呋喃妥因、磷霉素或SMZ-TMP
- 急性肾盂肾炎：喹诺酮类或头孢曲松

四、抗生素耐药性问题

当前主要耐药威胁：
- MRSA（耐甲氧西林金黄色葡萄球菌）：万古霉素、利奈唑胺
- VRE（耐万古霉素肠球菌）：利奈唑胺、达托霉素
- ESBL产酶菌：碳青霉烯类
- CRE（碳青霉烯耐药肠杆菌）：多粘菌素、替加环素、头孢他啶/阿维巴坦

抗菌药物管理（AMS）策略：
- 处方前评估：是否真正需要抗生素
- 48~72小时评估：根据临床反应和微生物学结果决定降阶梯或停药
- 促进口服序贯疗法：尽早从静脉给药转为口服给药
